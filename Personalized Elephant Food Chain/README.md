# Personalized Elephant Food Chain System

This project aims to develop a Personalized Elephant Food Chain Recommendation System.

## Features
- AI-based food recommendations
- Daily reporting
- Caretaker dashboard

## Philosophy: Gravity vs Antigravity

- **Gravity Model (Traditional)**: Fixed rules, same food for all, no adaptation. Rigid.
- **Antigravity Model (This System)**: Flexible, adaptive, data-driven. The system "floats above rigid rules" using AI to make intelligent daily decisions based on real-time data.
